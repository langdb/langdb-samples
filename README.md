# LangDB Samples

This repository hosts code samples for LangDB. All samples are written in SQL.

## Structure

The repository is structured as follows:

- `agents/`: This directory contains SQL samples related to agents.

## Agents

| Sample                                           | Description                                  |
| ------------------------------------------------ | -------------------------------------------- |
| [Basic RAG on PDF ](agents/pdf-rag.sql)        | Explains how to run RAG on PDF files         |
| [Extract Tables from PDF](agents/pdf-tables.sql) | Convert PDF documents into structured tables |
